## RECORDING PRODUCTION NOTES

### Setup
```
ZOOM CONFIGURATION:
- 3 participants (Consultant, Analyst, Scientist)
- Screen share: Meridian_ER_Presentation.pptx
- Cameras: ON (all 3 visible)
- Audio: USB microphones recommended
- Recording: Cloud + local backup

SLIDE ADVANCEMENT:
- Manual (speaker says when, presenter clicks)
- Practice run-through recommended
- Timing: Advance at natural pauses
```

### Pacing Guide
```
- Each speaker: 60-90 sec chunks (natural rhythm)
- Visual changes: Every 45 seconds (keeps attention)
- Key metrics: Display on slide while reading
- Emphasis: Slow down for numbers, speed up for narratives
- Vocal tone: Conversational (not robotic)
- Confidence: First takes usually best (natural energy)
```

### Audio Tips
```
- Record locally on each phone as backup
- Test levels: -10dB to -3dB (not peaking)
- Remove background noise in post-production
- Enunciate numbers clearly (Six point nine, not 6.9)
- Pause before key statistics (lets them sink in)
- No "um"s or "uh"s (easy to cut in editing)
```

### Post-Production
```
1. Export Zoom video + audio
2. Clean audio (Audacity): Noise reduction, normalize volume
3. Edit video (iMovie/CapCut): Sync cleaned audio, add slides
4. Color correct: Slightly boost skin tones
5. Export: MP4, 1080p, H.264
6. Final check: 7-10 min requirement, audio clear, visuals readable
```
